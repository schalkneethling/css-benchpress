Project Goal Summary

css-benchpress helps developers and tooling authors discover where realistic, framework-free CSS patterns become performance problems by growing web-platform fixtures until measurable regressions appear.

The current milestone is a Node.js and TypeScript CLI prototype that lists cases, runs a growth loop, collects Performance API and Chromium-specific metrics, detects threshold crossings, writes JSON and HTML reports, saves the smallest-scale repro, and ships an initial case corpus around selectors, custom properties, layout instability, and paint-heavy effects.
